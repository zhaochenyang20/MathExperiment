numpy.ma.dumps¶

numpy.ma.dumps(a)[source]¶

Return a string corresponding to the pickling of a masked array.

This is a wrapper around cPickle.dumps.

Parameters:
a : MaskedArray

The array for which the string representation of the pickle is returned.
